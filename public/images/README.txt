PLACEHOLDER IMAGES INSTRUCTIONS

This directory needs the following placeholder images:

1. default-avatar.png
   - A simple user silhouette icon for profile pictures
   - Recommended size: 200x200 pixels
   - Use a neutral gray background with a simple user silhouette

2. comic-placeholder.jpg
   - A placeholder for comic book covers
   - Recommended size: 400x600 pixels (2:3 aspect ratio)
   - Dark background with "No Cover Available" text

You can create these images using any image editor like Photoshop, GIMP, or an online tool like Canva.

Alternatively, you can use online placeholder services like:
- https://placehold.co/400x600/111827/ffffff?text=No+Cover (for comic covers)
- https://placehold.co/200x200/111827/ffffff?text=User (for user avatars)

These images are used as fallbacks when remote images fail to load.
